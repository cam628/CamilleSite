## ğŸ—ï¸ Architecture Overview

### **Framework: Next.js 15.5.5+ (App Router)**
- Uses the modern App Router architecture (not Pages Router)
- File-based routing: files in \`app/\` directory become routes
- Server Components by default for better performance

### **Language: TypeScript 5**
- Full type safety across the application
- Configured in \`tsconfig.json\`

### **Styling: Tailwind CSS 4.1.13**
- Utility-first CSS framework
- Configured via \`tailwind.config.ts\`
- Processed through PostCSS (\`postcss.config.mjs\`)

## ğŸ“„ File-by-File Breakdown

### **Configuration Files**

#### \`package.json\`
- **Purpose**: Defines project metadata, dependencies, and npm scripts
- **Key Scripts**:
  - \`npm run dev\` - Start development server
  - \`npm run build\` - Build for production
  - \`npm run start\` - Start production server
  - \`npm run lint\` - Run ESLint
- **Dependencies**:
  - \`next\`, \`react\`, \`react-dom\` - Core framework
  - \`tailwindcss\`, \`@tailwindcss/postcss\` - Styling
  - \`typescript\` - Type safety
  - \`eslint\`, \`eslint-config-next\` - Code quality

#### \`tsconfig.json\`
- **Purpose**: TypeScript compiler configuration
- **Key Settings**:
  - Target: ES2017
  - Module: ESNext
  - JSX: Preserve (Next.js handles JSX)
  - Path aliases: \`@/*\` maps to project root

#### \`next.config.ts\`
- **Purpose**: Next.js framework configuration
- **Current**: Default empty config (ready for customization)

#### \`tailwind.config.ts\`
- **Purpose**: Tailwind CSS customization
- **Content Paths**: Scans \`pages/\`, \`components/\`, and \`app/\` for class usage
- **Theme**: Default Tailwind theme (extendable)

#### \`postcss.config.mjs\`
- **Purpose**: PostCSS configuration for processing CSS
- **Plugin**: \`@tailwindcss/postcss\` - Processes Tailwind directives

---

### **Application Files**

#### \`app/layout.tsx\`
- **Type**: Server Component (default in App Router)
- **Purpose**: Root layout that wraps all pages
- **Features**:
  - Sets HTML metadata (title, description)
  - Imports global CSS
  - Provides \`<html>\` and \`<body>\` structure
  - All pages render as \`{children}\`

#### \`app/page.tsx\`
- **Type**: Server Component
- **Route**: \`/\` (home page)
- **Purpose**: Main landing page
- **Features**:
  - Responsive design (mobile-first)
  - Tailwind CSS styling
  - Welcome message and call-to-action cards

#### \`app/globals.css\`
- **Purpose**: Global stylesheet
- **Content**: Imports Tailwind CSS (\`@import "tailwindcss"\`)
- **Usage**: Imported in \`layout.tsx\` to apply globally
